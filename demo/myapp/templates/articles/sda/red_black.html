Arbori red-black

Ce sunt Arborii red-black ?

  • Arborii red-black sunt arbori binari de căutare care mai conțin o informație suplimentară, aceea fiind culoarea.
  • Prin restrângerea modului în care se colorează nodurile pe orice drum de la rădăcină la o frunză, arborii red-black garantează că nici un astfel de drum nu este mai lung decât dublul lungimii oricărui alt drum, deci că arborele este aproximativ echilibrat.
  • Fiecare nod este fie roşu, fie negru
  • Fiecare frunză (nil) este neagră
  • Dacă un nod este roşu, atunci ambii fii ai săi sunt negri
  • Fiecare drum simplu de la un nod la un descendent care este frunză conţine acelaşi număr de noduri negre.
  • Radăcina este tot timpul neagră.
  • Poză din "Introduction to Algorithms" by Thomas H. Cormen

Rotații pe arborii red-black

  • Deoarece operațiile de insert și delete pot strica structura arborelui, acesta se repară prin rotații.
  • Rotația la stânga / dreapta :
  • Implementare pentru RotateLeft :
    void RotateLeft(Node*& root, Node*& x){
        Node* y = x->right;
        x->right = y->left;
        if(y->left != nullptr)
            y->left->parent = x;

        y->parent = x->parent;
        if(x->parent == nullptr)
            root = y;
        else if(x == x->parent->left)
            x->parent->left = y;
        else
            x->parent->right = y;
        y->left = x;
        x->parent = y;
    }
