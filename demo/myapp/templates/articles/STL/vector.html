STL Vector

Ce este un Vector din STL ?

  • Vectorul din STL este un container dinamic, secvențial și liniar ce-și poate modifica dimensiunea in timpul execuției (alocare dinamică)
  • Vectorii își gestionează singuri alocarea memoriei. Când vectorul crește, alocă mai multă memorie, iar când se micșorează, deallocatează memoria după necesități.
  • Acesta se declară astfel : vector<tip_data> nume_vector(N,X);
  • N reprezintă numărul de elemente iar X valorea cu care se umple vectorul. Acestea pot să lipsească nefiind necesare.

Operații și funcții aplicate pe Vector

  • nume_vector.push_back(variabila) adaugă la finalul vectorului variabila introdusă intre paranteze.
  • nume_vector.pop_back() sterge de la finalul vectorului un element
  • nume_vector.size() returnează numărul de elemente ale vectorului
  • nume_vector.erase() - Șterge unul sau mai multe elemente din vector la poziția specificată. Aceasta implică mutarea tuturor elementelor de după poziția ștearsă pentru a le rearanja.
  • nume_vector.resize() - Redimensionează vectorul.
  • nume_vector.clear() - Elimină toate elementele din vector, dar nu modifică capacitatea acestuia
  • nume_vector.empty() - Verifică dacă vectorul este gol. Returnează true dacă vectorul este gol, altfel false.
  • begin() - Returnează un iterator la primul element din vector. Acest iterator poate fi folosit pentru a parcurge elementele începând cu primul element.
  • end() - Returnează un iterator care indică poziția "de după ultimul element" al vectorului. Acesta este utilizat pentru a marca sfârșitul unui interval de iterare.
  • vector<int>::iterator - Este un tip de iterator specific pentru un vector de tip int. Poate fi folosit pentru a traversa și modifica elementele vectorului în mod eficient.

Tipuri de parcurgere :

1) Cu iterator :

vector<int> a;
for(int i = 0 ; i < 5 ; ++i)///Penru a usura adaugam primele 5 numere in vector
    a.push_back(i);
for(vector<int>::iterator i = a.begin(); i != a.end() ; ++i)
    cout << *i;

2) Folosind cuvântul cheie "auto" care preia tipul de date automat :

for(auto i : a)
    cout << i;

3) Parcurgere standard :

for(int i = 0 ; i < a.size() ; ++i)
    cout << a[i];

Librăria <algorithm>

  • În C++, biblioteca algorithm conține un set de funcții standardizate pentru a efectua operații casupra containerelor, cum ar fi sortarea, căutarea, modificarea, și permutarea datelor. Ea face parte din Standard Template Library (STL) și este extrem de utilă pentru a simplifica și eficientiza manipularea colecțiilor de date.Aceasta este foarte utilizată pe vectorii din STL deoarece usurează scrierea mult, fiind totodata algoritmi eficienți.
  • std::sort(InputIt first, InputIt last, func) - Sortează elementele între iteratorii first și last având ca parametru (nu e obligatoriu) functia cu numele func ce setează criteriul de sortare.
  • std::find(InputIt first, InputIt last, const T& value) - Caută primul element egal cu value între iteratorii first și last.
  • std::binary_search(InputIt first, InputIt last, const T& value) - Verifică dacă value există într-o secvență sortată definită de iteratorii first și last. Aceasta se realizează in O(log2n)
  • std::find_if(InputIt first, InputIt last, UnaryPredicate p) - Caută primul element care îndeplinește condiția p între first și last.
  • std::reverse(BidirIt first, BidirIt last) - Inversează ordinea elementelor între iteratorii first și last.
  • std::fill(ForwardIt first, ForwardIt last, const T& value) - Umple secvența definită de iteratorii first și last cu valoarea value.
  • std::replace(InputIt first, InputIt last, const T& old_value, const T& new_value) - Înlocuiește toate aparițiile valorii old_value între iteratorii first și last cu new_value.
  • std::next_permutation(BidirIt first, BidirIt last) - Reordonează elementele între iteratorii first și last pentru a obține următoarea permutare lexicografică.
  • std::count(InputIt first, InputIt last, const T& value) - Numără aparițiile valorii value între iteratorii first și last.
  • std::accumulate(InputIt first, InputIt last, T init) - Agregă valorile între first și last, începând cu valoarea inițială init.
